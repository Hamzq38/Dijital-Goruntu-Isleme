# Dijital Görüntü İşleme Ödevi

Bu proje, Python ve PyQt6 kullanılarak kütüphanesiz (OpenCV kullanmadan) temel görüntü işleme algoritmalarını içermektedir.

## Özellikler
- Gri Seviye Dönüşümü
- YUV Renk Uzayı Dönüşümü
- Binary (Eşikleme)
- Histogram Eşitleme
- Kontrast Germe

## İçerik
- `arayuz.py`: Uygulamanın kaynak kodu.
